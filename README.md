#### 项目结构与文件组织

| 文件/目录 | 描述                             |
| --------- | -------------------------------- |
| /data     | 数据文件，保存输出文件的路径     |
| /model    | 命名实体识别模型                 |
| /ner      | 命名实体识别的相关代码           |
| /ontology | 构建本体及图谱转Jena规则的相关类 |
| /owl      | 本体及图谱得存放目录             |
| /util     | 工具类                           |
| main.py   | 主逻辑                           |
